# uplink
Note-taking CLI with search feature

### Installation

**For development**

1. Clone the repo
2. Create a virtualenv
3. Source the virtualenv
4. `pip install .`
5. You can now use the library like: `uplink --help`


**For deployment**

1. Clone the repo
2. `pip install .`
3. You can now use the library like: `uplink --help`
